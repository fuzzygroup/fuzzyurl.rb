    decompose_url(url) click to toggle source

    Given a URL (like `'example.com/some/path?args=1`), returns a hash containing protocol (like `'https'`), hostname (like `'example.com'`), port (like `443`), path (like `'/some/path?args=1'`), and split_path (like `['some', 'path?args=1']). Returns nil when a URL match is impossible.

def self.decompose_url(url)
  if m = url.match(%r{
                    (?: ([a-z]+)://)?             ## m[1] is protocol

                    (?: [a-zA-Z0-9]+              ## ignore auth info
                        (?: : [a-ZA-Z0-9]*)
                        @
                    )?

                    ([-a-z0-9\.\*]+)?             ## m[2] is hostname
                                                  ## match * too

                    (?: : (\*|\d+))?              ## m[3] is port
                                                  ## match * too

                    (/ .*)?                       ## m[4] is path and args
                   }x)
    protocol = m[1] ? m[1].downcase || 'http'
    hostname = m[2] ? m[2].downcase || nil
    port = m[3] || PORT_BY_PROTOCOL[protocol] || 80
    path = (m[4] || '/').split('/')

    { :protocol => protocol,
      :hostname => hostname,
      :port => port,
      :path => path }

  else ## no match
    nil
  end
end
